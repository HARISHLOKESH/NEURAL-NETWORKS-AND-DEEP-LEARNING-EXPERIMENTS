# 🧠 Neural Networks and Deep Learning (NNDL) Lab

## 📌 Overview

This repository contains implementations of various **Neural Networks and Deep Learning experiments** as per the NNDL lab syllabus.
Each experiment demonstrates real-world applications of AI using **TensorFlow, Keras, and Python**.

---

## 🎯 Objectives

* Understand fundamentals of Deep Learning
* Implement neural network models for real-world problems
* Analyze model performance using metrics and visualizations
* Gain hands-on experience with CNN, RNN, LSTM, and Transfer Learning

---

## 🧪 List of Experiments

### 1️⃣ Face Recognition Attendance System

* Uses SVM for face classification
* Automatically marks attendance

### 2️⃣ Fraud Detection using ANN

* Detects fraudulent transactions
* Handles class imbalance

### 3️⃣ MNIST Digit Classification

* Classifies handwritten digits
* Visualizes accuracy and loss

### 4️⃣ CNN Image Classification

* Classifies animal images
* Learns features like edges and shapes

### 5️⃣ RNN Temperature Prediction

* Predicts future temperature
* Analyzes memory effect of sequences

### 6️⃣ LSTM Autoencoder for Anomaly Detection

* Detects anomalies using reconstruction error

### 7️⃣ Object Detection (Smart Parking)

* Detects vehicles using CNN
* Uses bounding boxes

### 8️⃣ CIFAR-10 Classification

* Classifies 10 image categories
* Visualizes CNN filters

### 9️⃣ Pneumonia Detection (Transfer Learning)

* Uses MobileNetV2
* Works on chest X-ray dataset

### 🔟 Real-Time Object Detection (YOLO)

* Detects persons in video
* Displays FPS and confidence

### 1️⃣2️⃣ Time Series Forecasting (LSTM)

* Predicts future values
* Evaluates using MAE, RMSE

## ▶️ How to Run

1. Install dependencies:

```bash
pip install tensorflow keras numpy pandas matplotlib opencv-python kagglehub
```

2. Run any experiment:

```
Shift + Enter (In Collab)
```

3. Ensure datasets are downloaded automatically using KaggleHub.
